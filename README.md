# ChatGPT-API-Python
用Python和OpenAI开发的API实现简单的Q&amp;A

### Requirements

* openai >= 0.27.0

### How to use

* Install package
  ```
  pip install openai
  ```
* Run the script
  ```python
  python chatgpt.py
  ```

### API key

* get key from https://platform.openai.com/
* Click the avatar on the right upper conner, View API keys
